leaflet-tilelayer-mask

Leaflet-tilelayer-mask adds mask effect to tilelayer.

Demo

default
default behaviour, clip foreground tilelayer with round soft-edge mask.
custom
You may use custome mask image.

Usage

L.tileLayer.mask('http://www.finds.jp/ws/tmc/1.0.0/Kanto_Rapid-900913-L/{z}/{x}/{y}.png', {
	attribution : 'NIAES, NARO',
	maskUrl : 'star.png',
	maskWidth : 800,
	maskHeight : 800
}).addTo(map);
